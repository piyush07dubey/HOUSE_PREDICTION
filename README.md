# 🏠 House Price Prediction (Regression)

## 📌 Project Overview
This project builds a regression model to predict house prices using structured housing features.

The focus areas include:
- Data Cleaning & Missing Value Handling
- Feature Engineering
- Feature Transformation (Log transformation, Encoding, Scaling)
- Model Comparison
- Evaluation using RMSE & MAE
- Residual Analysis

---

## 📊 Dataset
House price dataset containing features such as:
- Overall Quality
- Living Area
- Garage Capacity
- Basement Area
- Year Built
- and more...

Target Variable:
- `SalePrice`
